Site para cadastro de clientes e fornecedores.

Crie um projeto de aplicação web para controle de dois cadastros: Clientes: com nome, cpf, rg, email, telefone e endereço; Fornecedores: com nome, razão social, cnpj, email, telefone e endereço; Os dados devem ser armazenados em memória. Utilize um HashMap ou um ArrayList para isso. A aplicação deverá permitir listagem, inclusão, alteração e remoção de registros.
